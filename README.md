# MediBridge

MediBridge is an AI-powered healthcare web application that helps users with:

## Features

- AI Symptom Checker (Gemini API)
- Medicine Information Search
- Doctor Guidance
- Appointment Booking
- Admin Dashboard
- User Login & Signup
- MySQL Database Integration

## Tech Stack

- HTML
- CSS
- JavaScript
- Node.js
- Express.js
- MySQL
- Gemini API

## Project Overview

MediBridge provides basic health assistance by analyzing symptoms, suggesting possible conditions, offering medicine information, and allowing users to book appointments.
Note: Backend APIs are configured for local development and require local Node.js server and MySQL setup.
